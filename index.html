Discors

Disors is a clone of the popular chat app Discord. Users are able to create and join servers, create text channels within servers and message each other through text channels. Users can also send direct messages or friend requests simply clicking on another user. Messages and notifications are sent and received in real time using WebSockets.

A Loop to the Past

A Loop to the Past is 2-D combat game inspired by The Legend of Zelda: A Link to the Past. The player fights of waves of enemies until they run out of life. A Loop to the Past uses JavaScript for game logic and HTML:5 Canvas for rendering.

Rails Lite

Rails Lite is a RubyGem that generates a MVC framework for building web applications. Some of the major features are SQLite or PostgreSQL ORM with associations and search, RESTful and custom routing, model level validations, CSRF protection, model/controller lifecycle methods and console commands.
